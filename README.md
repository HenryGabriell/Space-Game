# Jogo-Espacial
 
É um jogo simples 2D de uma nave no espaço. Objetivo é sobreviver o maximo de tempo possivel sem ser atingido por meteoros.
<br>
Desenvolvido para um estudo de caso da Faculdade.
<br>
Feito no Unity em C#.
<br>
